<h1 align="center">CSC 3210 - Computer Organization and Programming</h1>

This course explores the fundamentals of computer architecture, assembly language, and machine-level programming. Topics include principles of computer organization, binary and hexadecimal arithmetic, signed and unsigned number representations, memory organization, addressing modes, procedure calls and stack frames, floating point operations, and instruction encoding. The course provides hands-on experience writing assembly language programs to understand how high-level code translates to machine instructions. Additionally, the course covers architectural paradigms including CISC versus RISC design philosophies and parallel architecture models and programming techniques.
