<h1 align="center">CSC 4222 - Fundamentals of Cybersecurity</h1>

This class introduces the core principles of cybersecurity, focusing on protecting systems, networks, and data from modern threats. Topics include operating system security, network and web security, cryptography, malware, and distributed system vulnerabilities. The course explores common attack techniques, defense mechanisms, and ethical considerations, blending theoretical foundations with practical exercises. Emphasis is placed on understanding real-world security challenges, secure system design, and methods for identifying, analyzing, and mitigating cyber threats.
