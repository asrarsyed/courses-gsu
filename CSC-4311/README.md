<h1 align="center">CSC 4311 - Cloud Computing</h1>

This course explores the technologies and systems that support modern cloud computing platforms. Topics include virtualization, containerization, distributed file systems, cloud databases, big data processing frameworks, data streaming, graph processing, resource management, consensus protocols, and serverless computing. The class examines influential research and real-world cloud architectures through lectures, discussions, hands-on programming, and a semester-long project. The focus is on designing, deploying, and evaluating scalable distributed applications.
