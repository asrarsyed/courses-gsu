<h1 align="center">CSC 3350 - Software Development-CTW</h1>

This course explores the disciplined process of software development, emphasizing core principles of software engineering and system design. It covers object-oriented design, UML modeling, software architecture, design patterns, testing strategies, and refactoring techniques. Practical development work uses tools such as Java, SQL, Visual Studio Code, and database management systems. The course includes both individual assignments and a collaborative team project, with a strong focus on documentation, clear communication, and maintainable design. Industry-relevant practices and formal specifications are emphasized throughout.
